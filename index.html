Feathered Trails Beta

Beta Download

To download, follow these instructions:

  1. Click the button below, make sure to allow download permissions.
  2. Then, find the downloaded .apk file in your files app, it will be named something like v0.2.2.3.apk.
  3. Tap it and click install.
  4. Once the app is installed, open it and make sure to enable location permissions.
Feathered Trails is currently only available on Android

Live location map

View all your sightings

View all your species
